# PETalytics
### Included
- Notebook files for training and data processing
- Data .xlsx and .csv files for halo assay data (.csv files are faw from https://petadex.net/halo-assay/?tab=activity , .xlsx file is from my own google sheets processing in combination with the halo_data_processing.ipynb notebook used for merging and filtering the datasets)
- Proposal and progress report papers

### Not included: 
- Dataset of 18+ million protein sequences, found at https://huggingface.co/datasets/petadex/catalytic-orfs-90pid
- Read-only PETadex PostgreSQL database (used to match halo assay data to mutant amino acid sequences)
